# v1.1.0

## Исправлено
- **Бэкап hosts.** При применении / сбросе копия `hosts` сохраняется в папку **Загрузки**:  
  `Downloads\hosts_backup_YYYY-MM-DD_HH-mm.txt`. Путь выводится в сообщении об успехе.
- Перед Apply снимаются **конфликтующие** ручные строки Spotify.
- **UI:** отступ между спиннером и текстом «Применяю…» на кнопке применения.

## Изменено
- Версия **1.1.0**.

# v1.0.0

## Новое
- **Свой IP.** Можно вручную ввести IPv4, проверить доступность (TCP :443) и выбрать этот узел для записи в `hosts`.
- **Выбор узла.** Клик по доступному прокси в списке — он пойдёт в hosts вместо автовыбора «лучшего».
- **Предпросмотр** отражает выбранный (или лучший) узел.

## Изменения
- Версия **1.0.0** — стабильный релиз с актуальным функционалом.
- Сборки для **Windows** (x64 / arm64, Setup + portable), **Linux** (AppImage, deb) и **macOS** (dmg, zip).
- Улучшения надёжности: валидация IP, sandbox Electron, ротация бэкапов hosts, мониторинг активного узла.

## Установка
Скачайте пакет для своей ОС со страницы [Releases](https://github.com/swd3k/spotify-discord-fixer/releases) и запустите.

⚠️ Неофициальный проект, не связан со Spotify / Discord / GeoHide. Трафик перенаправляемых доменов Spotify (включая авторизацию) идёт через сторонний сервис GeoHide — используйте на свой риск.
