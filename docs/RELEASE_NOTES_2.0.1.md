## Что нового

### UI / UX
- Единое состояние кнопок **Применить** / **Сбросить** (без залипания)
- Спиннер «Применяю…», зелёный бейдж **Активно**
- Сворачиваемый блок «Как это работает»
- Модалка согласия: затемнение фона и **Esc**
- Компактная сетка, типографика, focus-ring

### Надёжность и обновления
- Баннер обновлений: поле `latest`, silent / Releases
- UAC: отмена, таймаут, проверка записи hosts
- Dismiss IP в `localStorage`
- Balloon трея один раз за сессию

### Репозиторий
- Документация **на русском**
- Удалён legacy Electron/Node с `main`

### Требования
- Windows 10 / 11  
- [.NET 8 Desktop Runtime](https://dotnet.microsoft.com/download/dotnet/8.0)  
- [WebView2](https://developer.microsoft.com/microsoft-edge/webview2/)  
- Права администратора для hosts  

### Быстрый старт
1. Установите Setup  
2. **Обновить** список узлов  
3. **Обновить и применить**  
4. Перезапустите Discord и Spotify
